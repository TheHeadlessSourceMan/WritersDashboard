WritersDashboard This project is a dashboard for managing multiple writing projects.

The vision is to have a little helper to say "your word count is behind on 'xyz' manuscript" or "you need to be working on richening settings for 'abc'"

Status:

  • Loads settings and projects, stages, and settings, from csv files.
  • Can define your own stages.
  • Estimates how much work you have left, ETA, and whether you are on track.
  • link back to actual writing project
  • indicate which projects are active and which are shelved
